Overview

Purpose

This site is an online store for handmade, unique, and colorful jewelry products. It aims to showcase products and provide an easy way to buy them online.

Audience

The site is meant to target women around 16-30 years old who gravitate towards colorful and vibrant fashion.

Dynamic elements

The main JavaScript features will be:

  • Filtering: clicking a filter button to see products based on price, category, or color.
  • Add product to cart: clicking a "add to cart" button will add it to an array for anything in the cart. I will also keep the amount of items in the cart counted visibly by the cart icon.
  • Search: typing in the search bar will filter products based on the name of the product.
  • Cart total: displaying the total cost of items in the cart.

    • Pseudocode - Computational Thinking

    1. Create an array of objects for each product with properties like name, price, category, color, and id

    2. Create another empty array to be ready to hold added products to the cart.

    3. Create function for rendering products list dynamically on each page.

    4. Create function for filtering products with conditional branching, add event listener for on click events per filter.

    5. Create function for adding to cart by finding the product in products array by id and pushing that to the cart array without changing the original product array. Also add click event listeners for the "add to cart" buttons.

    6. Create function to update the cart display in the sidebar depending on items in cart. If cart is empty, display message saying so.

    7. Create thumbnail images for each product to be in the cart vs how it sits on the product page.
